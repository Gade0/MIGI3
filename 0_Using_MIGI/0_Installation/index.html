Installation

  1. Remove any previous version of MIGI 2 along with the MIGI folder associated
  2. Download MIGI 3
  3. Make MIGI's executable an exception to your anti-virus if flagged
  4. Put it inside your CS:GO installation folder next to CS:GO's executable
  5. Run as admin the first time to build the symbolic links
  6. Run MIGI again but normally
